LeetCode Problem: Valid Anagram

Problem statement: Given two strings s and t, return true if t is an anagram of s, and false otherwise. An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

Example 1:


                Input: s = "anagram", t = "nagaram"
                Output: true

Example 2:


                Input: s = "rat", t = "car"
                Output: false

Solution:

  1. Create an unordered map that represents the frequencies of characters in both strings. It takes in a character as the key and an integer as the value.
  2. Iterate through each character c in s and for each character, increment its count in map.
  3. Iterate through each character x in t and for each character, decrement its count in map..
  4. Iterate through each value in map and check if any are non-zero.
  5. If any value is non-zero, return false. Otherwise, return true.

Code:


            public boolean isAnagram(String s, String t) {
                Map count = new HashMap<>();
        
                for (char x : s.toCharArray()) {
                    count.put(x, count.getOrDefault(x, 0) + 1);
                }
        
                for (char x : t.toCharArray()) {
                    count.put(x, count.getOrDefault(x, 0) - 1);
                }
                
                for (int val : count.values()) {
                    if (val != 0) {
                        return false;
                    }
                }
                
                return true;
            }

How to Write a Winning Resume

Resume Basics

The essential components of a resume include:

  1. Contact Information: Include your name, address, phone number, and email.
  2. Location: Specify your city and state.
  3. Professional Summary: Provide a one-paragraph summary highlighting your current position, key achievements, skills, and experience. For guidance, refer to Jobscan.
  4. Education: List your school name, degree, location, field of study, GPA, relevant coursework, and expected graduation date. Prioritize this section if you have limited work experience.
  5. Experience: Detail relevant work experiences, including company name, dates of employment, key achievements, skills demonstrated, and technologies used. Employ action words to describe your contributions, quantify achievements where possible, speak in the present tense for current roles, and follow the format: "Performed A task using B tool(s) to accomplish C purpose/goal in D amount of time, resulting in E outcome." For example, "Collaborated with 3 teammates to develop a website using R and JavaScript in 2 months that geographically visualizes the number of endangered species in King County for a class project, achieving high marks and praise from the professor."
  6. Skills:
    • Programming Languages: Python, Java, C/C++, JavaScript, Go, R.
    • Frameworks/Libraries: React, Node, AWS, NumPy, Pytorch, Flask, Django, Spark.
  7. Projects: Include project names, dates, your role, a description, technologies used, and your specific contributions. Apply the same principles as the work experience section.
  8. Extras: Add links to your GitHub, LinkedIn, portfolio website, and any relevant blogs.

Additional Resume Information

  • Utilize action words extensively to convey your accomplishments dynamically.
  • Ensure your resume is free from typos; even a single mistake can be detrimental.
  • Incorporate relevant keywords extensively, as many employers use automated systems for initial resume screenings. Use Jobscan to help tailor your resume.
  • Avoid fabrications on your resume; authenticity is key, and discrepancies can lead to disqualification.
  • Attend resume workshops and utilize available resources, such as the University of Washington's career services. Visit UW Careers for more information.
